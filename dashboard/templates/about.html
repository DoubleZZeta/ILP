About This Dashboard

Overview

This dashboard provides interactive visualizations and analysis of drone delivery logs. It leverages Python data analysis and machine learning techniques to extract insights from delivery data.

Features

  • KPIs: Total deliveries, active drones, service points, and average distance metrics.
  • Data Analysis: Multiple visualizations including elbow plots, service point breakdowns, drone stacking, and temporal patterns.
  • K-means Clustering: (Removed) Previously used to identify service zones; dashboard now shows suggested service points on the map directly.
  • Raw Data: Full access to the delivery log data in tabular format.

Technology Stack

  • Backend: Flask (Python web framework)
  • Data Processing: Pandas, NumPy
  • Visualization: Simple matplotlib PNGs (rendered server-side) to keep site easy to reproduce and inspect
  • Frontend: Bootstrap 5, Jinja2 templating

How to Use

  1. Home: View key performance indicators at a glance.
  2. Analysis: Explore multi-dimensional views of the delivery data.
  3. Clustering: (Removed) Clustering controls were removed to keep the project simple and reproducible. Suggested service points are shown on the map.
  4. Data: Download or review the raw delivery logs.

Data Source

This dashboard reads delivery data from delivery_logs.csv in the project root. The analysis functions are located in analysis/analysis.py. "Service points" shown are the active/used service points derived from delivery logs. The map view shows delivery locations and suggested new service points.
